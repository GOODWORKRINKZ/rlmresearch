<!-- Mimo API details (from RESEARCH.md) -->
Mimo is OpenAI-compatible:
```python
backend="openai"
backend_kwargs={
    "api_key": "MIMO_API_KEY",
    "model_name": "XiaomiMiMo/MiMo-V2.5-Pro",
    "base_url": "https://api.siliconflow.cn/v1",
}
```

<!-- RLM other_backends limitation -->
RLM only accepts exactly 1 other_backend. So the routing is binary:
- Primary (depth=0): one model
- Sub-calls (depth=1): one other model

For Mimo integration, we have two modes:
1. **Mimo as primary**: Mimo at depth=0, DeepSeek Flash at depth=1
2. **Mimo as sub-call**: DeepSeek Pro at depth=0, Mimo at depth=1
3. **Full Mimo**: Mimo at depth=0, Mimo at depth=1 (saves API costs if Mimo is cheaper)

Config-driven switching via `ACTIVE_PROVIDER=deepseek|mimo` env var.

   - `mimo_model: str` — loaded from `MIMO_MODEL` env var, default `"XiaomiMiMo/MiMo-V2.5-Pro"`
   - `active_provider: str` — loaded from `ACTIVE_PROVIDER` env var, default `"deepseek"`. Valid values: `"deepseek"`, `"mimo"`

2. Add validation in `__post_init__`:
   - If `active_provider == "mimo"` and `mimo_api_key` is empty, raise ValueError with helpful message
   - Validate `active_provider` is one of ("deepseek", "mimo")

3. Add new property `active_backend_kwargs`:
   ```python
   @property
   def active_backend_kwargs(self) -> dict:
       if self.active_provider == "mimo":
           return {"api_key": self.mimo_api_key, "model_name": self.mimo_model, "base_url": self.mimo_base_url}
       return self.rlm_backend_kwargs  # DeepSeek Pro
   ```

4. Add new property `active_other_backend_kwargs`:
   ```python
   @property
   def active_other_backend_kwargs(self) -> list[dict]:
       if self.active_provider == "mimo":
           # Mimo as primary, Mimo Flash as sub-call (or same model)
           return [{"api_key": self.mimo_api_key, "model_name": self.mimo_model, "base_url": self.mimo_base_url}]
       return self.other_backend_kwargs  # DeepSeek Flash
   ```

5. Add property `active_model_name` for logging:
   ```python
   @property
   def active_model_name(self) -> str:
       if self.active_provider == "mimo":
           return self.mimo_model
       return self.deepseek_pro_model
   ```

6. Update `get_settings()` log to show `active_provider` and `active_model_name`.

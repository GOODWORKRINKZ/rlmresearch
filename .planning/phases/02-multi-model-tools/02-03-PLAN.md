<!-- VS Code tool calling problem -->
VS Code sends requests with extra fields: `tools: [...]`, `tool_choice: "auto"`. The ChatCompletionRequest Pydantic model currently ignores unknown fields (extra="ignore" default). But if we later use strict parsing, or if the model sees tool definitions in context, RLM gets confused.
